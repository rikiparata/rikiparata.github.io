What is Grid Based Design?

The grid based design is a layout designed to take full advantage of our natural way of reading websites. Our eyes naturally begin at the top left and work their way across the website title, and then down to a photo next to a paragraph title, followed by the paragraph itself.

Whats all the hype about responsive webpages?

Using responsive webpages is great beacause it ensures that no matter the device or screen size you use to access the website, it will resize automatically and make your webpage look great to the reader.

What is semantic structure?

Semantic structure is the HTML code we write that can be understood by both the browser and the people reading it. Examples of this include using "h1" to define you webpage's header or title.

Any other points you found interesting and want to reflect on.
